# TP Microservice

## Overview

This repository contains a microservice-based application.

## Features

- Microservice architecture
- Scalable components
- Easy deployment

## Installation

```bash
git clone https://github.com/macfim/tp-microservice.git
cd tp-microservice
npm install
```

## Usage

```bash
npm start
```
